# Projeto

## Status
Projeto em desenvolvimento.

## Descrição
Projeto utilizado para prática de JavaScript com manipulação do DOM e lógica de carrinho de compras.

## Observação
A estrutura inicial (HTML, CSS e arquivos de mídia) já estava pronta.  
O foco deste projeto é a implementação da lógica em JavaScript, especialmente a manipulação de formulários, leitura de dados de uma combobox, cálculo de valores e atualização dinâmica da interface.

## Tecnologias
- HTML
- CSS
- JavaScript

## Execução
Abra o arquivo `index.html` em um navegador.
